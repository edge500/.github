# Off By One

Small fixes. Sharp tools. Better systems.

We build open-source projects for developers, builders, and people who care about thoughtful software.

## What We Care About

Off By One is a small open-source organization focused on practical tools with a human edge:

- AI and developer workflows
- local-first productivity
- clean documentation
- small utilities that remove daily friction
- experiments that can grow into serious products

The name comes from a classic programming mistake: the tiny edge case that changes everything. We like that idea. Good software is often made by noticing the small thing everyone else skipped.

## Current Direction

We are shaping this org around:

- **SummitMind**: local-first memory and reflection tools
- **developer experience**: workflows that make building less painful
- **open-source learning**: beginner-friendly issues, clear docs, and visible progress
- **AI systems**: practical experiments with agents, RAG, evals, and safer automation

## How To Join

We welcome people who want to contribute with care.

You can help by:

- improving docs
- testing projects
- opening issues with clear examples
- designing better user flows
- fixing small bugs
- suggesting useful tools we should build

Good contributors do not need to be loud. They need to be honest, curious, and consistent.

## Working Style

- Keep things simple before making them clever.
- Explain the problem before proposing the fix.
- Respect user privacy.
- Prefer useful projects over shiny demos.
- Leave the codebase easier to understand than you found it.
