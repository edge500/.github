# Contributing To Off By One

Thanks for wanting to help.

Off By One is built around small, useful contributions. You do not need to ship a huge feature to matter here.

## Good Contributions

- Clear bug reports.
- Better docs.
- UI polish.
- Small fixes with simple explanations.
- Tests for behavior that could break.
- Ideas that solve a real problem.

## How We Work

1. Start with the problem.
2. Keep the change focused.
3. Explain your thinking in plain language.
4. Respect user privacy.
5. Leave the project easier to understand.

## Before Opening A Pull Request

- Make sure the change is related to the issue or project goal.
- Check the app still runs.
- Avoid unrelated formatting churn.
- Include screenshots for visible UI changes.

## Tone

Be kind, direct, and useful. We are here to build, learn, and help each other get sharper.
